# 今日の一歩 — ワンオラクル（仮）

タロットを1枚引いて、今日の小さな迷いに「テーマ／問いかけ／背中を押す一言／小さな次の一歩」を返す、
**ブラウザだけで完結する**静的 Web アプリです。

立ち位置は **エンタメ・自己理解**。的中を保証するものではありません。

## 使い方
[https://issuenuts.github.io/web-uranai/](https://issuenuts.github.io/web-uranai/) を開いて、「カードを1枚引く」だけ。
ローカルで動かすなら `index.html` をブラウザで開くだけ（サーバー不要・ビルド不要）。

## 大切にしていること
- **個人情報を持ちません**。入力したひと言も引いた結果も、その端末のブラウザ内だけで処理し、送信・保存しません。
- **引き方は透明**。大アルカナ22枚 × 正位置／逆位置を、単純な乱数で公平に1枚。
- **煽りません**。凶札も「切り替え・気づき」として前向きに読みます。深刻な悩みは専門の窓口を案内します。
